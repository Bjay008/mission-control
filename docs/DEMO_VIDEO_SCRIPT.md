# Demo Video Script

## Target

Length: 75–90 seconds. Record the real application at 1440×900 or higher. Keep the cursor deliberate and avoid cuts during pipeline execution.

## Shot list and voiceover

### 0:00–0:08 — The problem

Show the full Mission Control launch screen.

Voiceover: “AI can generate content. The hard part is coordinating an entire mission without losing context, ownership, or momentum.”

### 0:08–0:18 — The command

Hold on the command field containing `Create Day 32.`

Voiceover: “Mission Control gives that work a CEO Brain. One creator command starts the frozen production company.”

### 0:18–0:52 — The memorable run

Select **Create Episode** once. Do not cut. Let CEO, Research, Script, Devotional, Voice, Visual, Thumbnail, SEO, QC, and Upload Package complete live.

Voiceover: “CEO Brain assembles the specialists, routes every accepted artifact, records every handoff, and keeps mission health current.”

### 0:52–1:08 — Governance

Scroll just enough to show 100% progress, mission health, and the activity or decision log.

Voiceover: “The interface is not animating a separate story. Every green state comes from the same validated orchestration engine.”

### 1:08–1:20 — The handoff

Return to the **Ready for YouTube** panel and `upload-package.json`.

Voiceover: “Mission Control finishes the package and stops before the creator-controlled external action. This is not another AI chat. It is Mission Control for work that must actually get finished.”

## Recording checklist

- Reset the demo before recording.
- Confirm the command reads `Create Day 32.`
- Hide notifications and unrelated browser tabs.
- Record one continuous pipeline run.
- Use the browser test before the take.
- Keep the deterministic demo disclosure visible.
- Export 1080p H.264 and retain the original recording.
